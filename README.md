# 다음책 (가칭)

공공도서관 대출 데이터 기반 도서 추천 웹 서비스.
완독 직후·도서관 방문 전, 내 도서관에서 **지금 빌릴 수 있는** 책을 이유와 함께 추천한다.

## 기술 스택
- Backend: Django, Django REST Framework
- Frontend: Vue
- Data: 도서관 정보나루 · 알라딘 · 네이버 도서 API (fixtures)

작성 예정.
